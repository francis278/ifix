Documentation File Save Dialog Box

You may simply enter the name of the file name you wish to save in the File Name box on the left, or select the file name from an existing file in your system. Once you have entered or selected the name, click on the OK Button to save the file. Press the CANCEL button to Close the Dialog & cancel the File Save operation.

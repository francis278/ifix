Memory

My-T-Soft uses Windows Global Memory for various things. My-T-Soft will operate faster if Global memory is physical system memory, and not swapped to disk as virtual memory. Although My-T-Soft will run in 2 MB systems, operating at 4 MB, 8 MB, or higher is recommended.

It is normal for the My-T-Soft System Information Mem:????K display to drop as you type on My-T-Soft, or move over the My-T-Soft display. You may also note System Resources change as My-T-Soft is used.
